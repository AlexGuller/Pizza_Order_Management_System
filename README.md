# Pizza_Order_Management_System
Modular pizza ordering system implemented in C on Linux.

This project is a **pizza ordering system** implemented in **C** on Linux, using dynamic memory allocation and pointers.

## Highlights
- Simulates a pizza shop ordering system with modular design
- Uses **multi-level pointers** and **dynamic memory management**
- Built with a **Makefile** to automate compilation
- Debugged with **GDB** and **Valgrind** to ensure memory safety
